## Installation

```bash
yarn install
```

## Running the server on port 3000

```bash
# development (watch mode)
yarn start

# production
yarn start:prod
```

## Generating `frontend/src/api` code

```bash
yarn gen
```

## Test

```bash
# unit tests (watch mode)
yarn test

# e2e tests
yarn test:e2e

# test coverage
yarn test:cov
```
